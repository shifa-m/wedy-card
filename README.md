# 💌 Wedding Card Web App

A modern and responsive web application for browsing and selecting beautiful wedding invitation cards.

---

## ✨ Features

* 🏠 Home page with latest wedding card designs
* 🎨 Filter options (Traditional, Modern, Luxury)
* 🛒 Cart page for selected items
* 👤 Profile page
* 🔍 Responsive design (Mobile + Laptop)
* ⚡ Smooth navigation using React Router

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* 🔀 React Router DOM
* 🎯 JavaScript (ES6+)

---
